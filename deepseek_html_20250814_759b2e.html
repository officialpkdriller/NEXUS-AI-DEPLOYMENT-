FREQUENTLY ASKED QUESTIONS

How do I deploy NEXUS-AI on my own server?

You can deploy NEXUS-AI on various platforms including Heroku, Railway, Render, and Replit. We provide one-click deployment options for each platform. Simply click on the corresponding deployment button above and follow the instructions. For advanced users, you can also clone the repository and run it locally or on your own server.

Is NEXUS-AI free to use?

Yes, NEXUS-AI is completely free and open-source. However, some advanced AI features may require API keys from third-party services (like OpenAI for GPT-4) which might have their own usage limits or costs. The core functionality of the bot remains free to use without any restrictions.

How do I customize the bot's commands and responses?

NEXUS-AI offers extensive customization options through configuration files. You can modify the config.js file to change default settings, add/remove commands, and customize responses. The bot also supports plugins that allow you to extend its functionality without modifying the core code.

Is my data secure with NEXUS-AI?

Absolutely. NEXUS-AI follows strict privacy guidelines and does not store your personal data or chat history. All conversations are processed in real-time and not persisted. When using AI features, messages are only sent to third-party APIs when necessary and are not stored by our systems.

How can I contribute to the project?

We welcome contributions from the community! You can contribute by submitting pull requests on GitHub, reporting issues, suggesting new features, or improving documentation. Please visit our GitHub repository for contribution guidelines and code of conduct.
